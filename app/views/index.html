Carregar ficheiro oficial de resultados

Arraste ou clique para selecionar o arquivo HTML (D_MEGA.HTM) ou formato CSV obtido do site de downloads da Caixa.

Como descarregar a base de dados oficial completa?

  1. Aceder ao portal oficial das Loterias Caixa.
  2. Ir para a secção de downloads ou estatísticas da Mega-Sena.
  3. Localizar o link "Download de Resultados" que fornece o arquivo no formato compactado zip contendo o HTML histórico completo.
  4. Extrair o arquivo e carregar o arquivo D_MEGA.htm (ou similar CSV) nesta aplicação.

Sincronização Ativa

Os dados carregados serão salvos em memória local no navegador. Nenhum dado é enviado para servidores externos, respeitando integralmente a confidencialidade e privacidade.
